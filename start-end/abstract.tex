\begin{Abstract}
    \begin{changemargin}{1cm}{1cm}
        This thesis presents the development of an embedded computer vision system for the real-time acquisition, detection, and analysis of bakery products, operating in an edge environment without network connectivity.\\
        The system is based on NVIDIA Jetson Nano DK and Raspberry Pi 5 platforms and integrates an RGB-D Microsoft Kinect sensor and a Raspberry Pi Camera Module 3 Wide deployed at two acquisition sites, enabling the collection of both two-dimensional and three-dimensional data.
        
        The acquired images are processed by a real-time detection model for the automatic identification of bread products. Following the detection stage, descriptive features are extracted at different phases of the production process: during the dough stage, through point cloud analysis for volume estimation and geometric characterization; during the finished product stage, through the evaluation of anomalies using an anomaly detection model trained on baked bread.
        
        The extracted features are aggregated into a structured dataset and used to model the statistical distributions of product characteristics, enabling a quantitative description of the variations observed throughout the production process.
        
        The proposed approach demonstrates the effectiveness of integrating computer vision, deep learning, and 3D acquisition techniques within embedded systems for monitoring and data-driven analysis in industrial environments.

\end{Abstract}
